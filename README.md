# 하루, 사진
> **사진과 기억나는 음악을 기록할 수 있는 App**
>
> 프로젝트 기간 : 2022.11.14 ~ 2022.12.17

#### Commit Message Convention
[docs]: 프로젝트 문서 작업 혹은 수정하는 경우
[add]: 코드 추가
[feat]: 새로운 기능 구현
[chore]: 코드 수정
[style]: UI 관련 기능
[move]: 프로젝트 내 파일 이동
[rename]: 파일 이름 변경하는 경우
[remove]: 파일을 삭제하는 경우
[fix]: 버그를 고쳐야 하는 경우
[hotfix]: 급하게 치명적인 버그를 고쳐야 하는 경우
[test]: 테스트 코드 작성하는 경우
[improve]: 향상이 있는 경우
